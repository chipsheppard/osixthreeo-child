OsixthreeO Child is a child theme for the OsixthreeO WordPress theme.

== Description ==

OsixthreeO is a great selection for any website. Create your own unique design with a flexible, lightweight theme that provides font, color, layout and many other controls via the WordPress customizer.

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in OsixthreeO in the search form and press the 'Enter' key on your keyboard.
3. Install and activate OsixthreeO Child.
5. Go to https://osixthreeo.com for a guide on how to customize this theme.
