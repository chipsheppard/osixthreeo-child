OsixthreeO Child is a child theme for the OsixthreeO WordPress theme.

== Description ==

OsixthreeO is a great selection for any website. Create your own unique design with a flexible, lightweight theme that provides font, color, layout and many other controls via the WordPress customizer. This theme is perfect for making your own customizations.

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button. Search for OsixthreeO and download the theme.
2. In your admin panel, go to Appearance -> Themes and click the 'Add New' button then click the 'Upload Theme' button and upload the osixthreeo-child.zip file.
3. Or FTP the osixthreeo and the osixthreeo-child themess into your /wp-content/themes directory.
4. Activate OsixthreeO Child.
5. Go to https://osixthreeo.com for a guide on how to customize this theme.
